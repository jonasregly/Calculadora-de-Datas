    Portfólio

    Este projeto é um challange nível Júnior do site front-end Mentor. Este desafio consiste em criar uma calculadora de idade que calcule os dias,os meses e os anos de uma determinada data inserida na página. Trabalhar com datas é algo simples, porém desafiador, pois requer uma atenção especial nas conversões necessárias a se fazer para um cálculo assertivo da página web. Este projeto ainda não esta, finalizado, a parte responsiva está funcionando,mas ainda necessita de alguns ajustes visuais para que o projeto seja finalizado. Este projeto foi feito com HTML,CSS e JavaScript.

    Este projeto consiste num relógio digital interativo na qual muda a foto de fundo e as mensagens de Boas-vindas de acordo com o momento do dia(manhã, tarde ou noite). É um projeto simples, que usa HTML,CSS e JavaScript apenas, mas que apesar de simples me deixou muito orgulhoso, pois foi o primeiro projeto que eu fiz. Foi bom de mais sentir aquela sensação de progressão conforme eu fazia linha por linha do código.
